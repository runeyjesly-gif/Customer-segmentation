# 🛍️ Customer Segmentation using Machine Learning

## 📌 Project Overview

Customer Segmentation is a Machine Learning project that groups customers into different clusters based on their purchasing behavior and demographic information. This helps businesses understand customer patterns and develop targeted marketing strategies.

This project was developed using the **Mall Customers Dataset** and multiple clustering algorithms to analyze customer behavior.

---

## 🎯 Objectives

- Perform customer segmentation using Machine Learning.
- Analyze customer purchasing behavior.
- Compare clustering algorithms.
- Visualize customer groups for business insights.

---

## 📂 Dataset

**Dataset:** Mall Customers Dataset

### Features

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🤖 Machine Learning Algorithms

- K-Means Clustering
- Hierarchical Clustering
- Gaussian Mixture Model (GMM)

---

## 📊 Project Workflow

1. Import Dataset
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. K-Means Clustering
6. Hierarchical Clustering
7. Gaussian Mixture Model (GMM)
8. Cluster Evaluation
9. Data Visualization

---

## 📈 Results

The project successfully segmented customers into meaningful groups based on Annual Income and Spending Score.

The Elbow Method was used to determine the optimal number of clusters.

Different clustering algorithms were compared to understand customer behavior more effectively.
